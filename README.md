# CS2 Skin Price Predictor (Mock Data Example)

## Описание проекта

Этот проект представляет собой модель машинного обучения для предсказания цен на скины Counter-Strike 2. Он демонстрирует полный цикл работы с данными: от (мокового) сбора и хранения до обработки, обучения модели и предсказания.

**Обратите внимание:** Для упрощения и возможности запуска проекта без сложной настройки, первичный сбор данных реализован с помощью **мокового генератора данных**. В реальном приложении этот модуль `scraper.py` был бы заменен на полноценный веб-скрепер для платформ типа BUFF163, Lisskins или Steam Community Market, что требует обхода анти-бот-систем, CAPTCHA и использования прокси.

### Цель

*   Собрать (моковые) исторические данные о ценах и характеристиках скинов CS2.
*   Хранить данные в локальной базе данных SQLite.
*   Выполнить исследовательский анализ данных (EDA) и инженерию признаков.
*   Обучить модель машинного обучения (Random Forest Regressor) для предсказания цен.
*   Оценить производительность модели.
*   Предоставить простой интерфейс для предсказания цен на новые скины.

## Используемые технологии

*   **Язык:** Python 3.x
*   **Сбор данных:** (Моковый генератор), `pandas`
*   **Хранение данных:** `sqlite3`
*   **Анализ и обработка данных:** `pandas`, `numpy`, `scikit-learn`
*   **Машинное обучение:** `scikit-learn` (Random Forest Regressor)
*   **Сохранение/загрузка моделей:** `joblib`
*   **Визуализация:** `matplotlib`, `seaborn` (используется в `data_processor.py` для EDA)

## Установка и запуск

### 1. Клонирование репозитория

```bash
git clone https://github.com/DizzyZ7/CS2_Skin_Price_Predictor.git
cd CS2_Skin_Price_Predictor

2. Создание виртуального окружения и установка зависимостей

Рекомендуется использовать виртуальное окружение для изоляции зависимостей.

python -m venv venv
# Активация виртуального окружения
# Windows
.\venv\Scripts\activate
# macOS/Linux
source venv/bin/activate

pip install -r requirements.txt
3. Генерация моковых данных и инициализация базы данных

Этот шаг сгенерирует синтетические данные и сохранит их в data/cs2_skin_prices.db.

python src/scraper.py
4. Обучение модели

Этот скрипт загрузит данные, обработает их, обучит модель и сохранит ее в models/price_predictor_model.pkl.

python src/model_trainer.py
5. Использование модели для предсказания

Вы можете предсказать цену скина, указав его характеристики:

python src/predictor.py \
    --name "AK-47 | Redline" \
    --wear "Field-Tested" \
    --float 0.20 \
    --stattrak False \
    --rarity "Covert"
Пример вывода:

Предсказанная цена для AK-47 | Redline (Field-Tested) Float 0.20 (StatTrak: False, Rarity: Covert): $25.50
Структура проекта

src/scraper.py: Генерирует моковые данные о ценах скинов и сохраняет их в базу данных.
src/database.py: Содержит функции для работы с базой данных SQLite (создание таблиц, вставка данных).
src/data_processor.py: Загружает данные из БД, выполняет EDA, очистку и инженерию признаков.
src/model_trainer.py: Обучает модель предсказания цен на основе обработанных данных и сохраняет её.
src/predictor.py: Загружает сохранённую модель и предоставляет интерфейс для получения предсказаний.
src/utils.py: Вспомогательные функции, такие как маппинг категориальных признаков.
data/: Директория для файла базы данных cs2_skin_prices.db.
models/: Директория для сохранённой модели price_predictor_model.pkl.
